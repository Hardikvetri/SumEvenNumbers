**Description:**
This C# project calculates the sum of even numbers in a given array. It demonstrates basic array manipulation and conditional logic in C#. The project includes functionality to initialize the array, filter out even numbers, and compute their sum.

**Features:**
- Initialize an array with integer values.
- Identify and sum all even numbers in the array.
- Output the result.
